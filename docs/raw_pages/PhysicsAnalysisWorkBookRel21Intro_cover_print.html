Introduction

The purpose of this Physics Analysis Workbook is to teach people in ATLAS about how to do a complete physics analysis. Each of the sections gives information about specific topics, and links to expert documentation when available, and where appropriate example scripts are provided.

Other workbooks that might be of interest are:

  • ATLAS Computing Workbook: Gives the basics of the software setup in the experiment
  • Software Development Workbook: for more information about software development

Below are some prerequisites that you should have knowledge of before continuing to the next sections.

Software

You will need to have some programming knowledge to do a physics analysis. These days this typically involves coding in C++ and/or python, so it would be good to be familiar with one or the other (or both!). Most users do their analysis on xAODs or ntuples using ROOT, with ROOT you can navigate these files interactively, or by writing C++ and/or python code.

C++

  • There are many books and online resources learning C++, for example: http://www.cplusplus.com
  • If you are looking for the meaning of a particular thing, e.g., map, you can use the search box at the top of http://www.cplusplus.com/[this webpage]]. This will bring you to a top page for map. Clicking on the map link will bring you to a detailed explanation.

Python

  • There are many python tutorials available, here are a few:
    • ATLAS software python tutorial: Software Tutorial: Python
    • Python tutorial: https://gist.github.com/kratsg/b7d33b1f0f0d253eafc7
    • Interactive Python tutorial: http://www.learnpython.org/
    • Visualize Python: http://pythontutor.com/
    • Python in ATLAS: https://indico.cern.ch/event/52191/

ROOT

  • Main ROOT (methods for plot-making and statistical analysis in C++ or python) webpage to get you started: https://root.cern.ch//
  • If you are new to ROOT you can use these resources:
    • tutorials
    • How To
    • User's guide
    • Root Forum
  • resources for python + ROOT:
    • Jupyter notebook: How to: ROOT notebooks, How to create a rootbook
    • rootpy
    • root_numpy

And finally, Google is great for getting help with your C++, Python, and ROOT questions!

Other resources

  • Software Tutorial : A great up-to-date tutorial that covers the latest recommendations for computing in ATLAS
  • The ATLAS detector paper
  • CERN document server to search for internal notes
  • For additional help you can subscribe to some general purpose e-groups:
    • Click "Browse existing groups and categories" (near the bottom of the page) to search for groups of interest to you
    • You can subscribe to these e-groups so emails will be sent you automatically from the group
    • For many groups there is also an archive of past discussions
    • e-groups of interest:
      • analysis software tools: hn-atlas-PATHelp@cern.ch
      • offline software help: hn-atlas-offlineSWHelp@cern.ch
      • Grid related tools: hn-atlas-dist-analysis-help@cern.ch
  • You can send your questions to those egroups or any other that you find relevant to your question. And don't be shy - people are happy to help you!

There is a lot of documentation out there ... the difficulty is finding it and then finding the version that is up-to-date (that's what this workbook will help you do!)
