ATLAS Analysis Software Group (ASG)

Convener(s): Alison Lister, Lukas Alexander Heinrich

THIS PAGE IS NO LONGER MAINTANED - PLEASE SEE AnalysisModelgroup INSTEAD! The Analysis Software Group (ASG) replaced Physics Analysis Tools (PAT) in the summer 2013. The group aims to develop C++/Python based tools appropriate for Physics Analysis. The group is responsible for the Atlas Analysis Model and for the development of class libraries (in C++ and Python), tools and procedures for physics analysis both in a batch and interactive environment. The complete mandate of ASG can be found here: ASG Mandate

The old PAT page can be found here: PhysicsAnalysisToolsLegacy . The new Analysis Model Group page is developing here.
