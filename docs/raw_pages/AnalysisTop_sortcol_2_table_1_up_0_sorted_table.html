AnalysisTop

AnalsyisTop is the standard analysis software for run-2 analysis in the Top Working Group. A list of available resources is available below: we suggest newcomers to start with one of the introductory tutorials, and to read very carefully the Start Guide. In case of questions, please send an e-mail to atlas-phys-top-topreco-atmanagers. For more information on all topics related to software & tools for analyses in the top group, refer to the TopReconstructionGroup twiki page.
