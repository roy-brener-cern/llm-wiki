Physics Analysis Workbook: Release 21

Examples discussed here use data formats produced in Athena release 21 (data17, reprocessed data15+data16, mc16).

An alternative source of information is: SoftwareTutorial

Chapter 1: Introduction to the Physics Analysis Workbook

An introduction to the workbook, along with some useful references for software basics to get you started.
  • Introduction to the Physics Analysis Workbook
  • Software resources
  • Other resources and getting help

Chapter 2: Computing Setup

The basic computing setup necessary to do analysis using Athena or the Analysis Releases, and getting started on the Grid. These sections are also references in later chapters of the workbook.
  • Introduction
  • Account setup
  • Analysis Release setup
  • Setting up ROOT
  • Setting up to use the Grid

Chapter 3: Data Formats

An overview of the different datasets, their associated formats, and how you go about finding and getting these datasets for analysis.
  • Overview of ATLAS data formats (RDO, HITS, ESD, xAOD, DxAOD)
  • The xAOD
    • Containers and key names
    • Accessing variables
  • What DxAODs exist?
  • File naming conventions
  • Tools for finding datasets
  • Available datasets

Chapter 4: All about the ATLAS data

Everything you wanted to know about the real ATLAS data.
  • Introduction
  • ATLAS detector operation
  • Trigger
  • Data Quality
  • Good Runs List
  • Luminosity
  • Pileup
  • (Re)Processing

Chapter 5: All about the ATLAS Monte Carlo

Everything you wanted to know about Monte Carlo simulations in ATLAS.
  • Introduction to ATLAS Monte Carlo simulation
    • The basics
    • Generator filters
    • Generator weights
    • Fast versus full simulation
    • Generating your own private Monte Carlo
  • Finding information about official Monte Carlo
  • Finding Monte Carlo cross-sections and filter efficiencies

Chapter 6: Combined performance tools for physics analysis

Recommendations from combined performance and data preparation groups of tools and methods that should be considered when doing your physics analysis. The information is presented in a table format with links to the expert material for more information.

Chapter 7: Analysis Examples

  • Software Tutorial: AnalysisBase example using EventLoop
  • Software Tutorial: AthAnalysis example using Athena
  • (to come: complete physics example)

Extra: Tips for migrating to Release 21 (from Release 20)

For people who have done analyses in Release 20, this chapter will provide tips and references for migrating your analysis to Release 21.
